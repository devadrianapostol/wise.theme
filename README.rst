==========
wise.theme
==========

Plone Theme for WISE

Features
--------

- Can be bullet points


Installation
------------

Install wise.theme by adding it to your buildout::

    [buildout]

    ...

    eggs =
        wise.theme


and then running ``bin/buildout``
